## Questions & Answers

### 1. Difference between getElementById, getElementsByClassName, querySelector?

getElementById → select one element by id  
getElementsByClassName → select multiple elements by class  
querySelector → select first matching element  
querySelectorAll → select all matching elements

### 2. How to create and insert element in DOM?

We use document.createElement() and appendChild().

### 3. What is Event Bubbling?

Event moves from child element to parent element.

### 4. What is Event Delegation?

Using parent element to handle events of child elements.

### 5. Difference between preventDefault() and stopPropagation()

preventDefault() → stop default browser action  
stopPropagation() → stop event bubbling
